# Desafio 01 do curso FullCycle
Criar uma imagem docker rodando um script em golang

## Requisitos
 - Ao rodar o container, deve printar no console 'Code.education Rocks!'
 - A imagem deve ter menos de 2mb
 - Publicar a imagem no docker hub

### Comando para rodar o container
```bash
docker run --rm -it --name golang md91/golang-fullcycle
```

[Link da imagem no docker hub](https://hub.docker.com/repository/docker/md91/golang-fullcycle/general)
